WEB DEVELPMENT SKILL

Learn HTML

HTML stands for Hyper Text Markup Language. HTML is the content of a web page. Examples of HTML include heading, paragraph, images, links, lists etc.

Check out the HTML Tutorial on w3schools.com.

Learn CSS

CSS stands for Cascading Style Sheets. CSS is the style of a webpage. Examples of CSS are font styles and colours, background colour borders, dimensions, layout etc.

Check out the CSS Tutorialon w3schools.com.

Learn Javascript.

JavaScript is the programing language of the web. JavaScript helps to make web pages Interactive and dynamic. Some common JavaScript tasks are:

  • Add new HTML to page.
  • Change the existing content/style.
  • React to user events (mouse clicks, key presses etc.)

Check out the Javascript tutroialon Javascript.info.
